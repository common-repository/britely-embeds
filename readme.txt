This plugin extends the shortcode syntax to insert Britely brites in any content using [britely=URL width=SIZE].

== Description ==

To find or create your own brites to embed, visit <a href="http://www.britely.com/">britely.com</a>!

== Installation ==

1. You can use the automatic plugin installer or your FTP program to upload this plugin to your wp-content/plugins directory.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Find the brite you want to embed at http://www.britely.com
1. Click the "share" button at the bottom right of the brite and select "WordPress"
1. Copy the shortcode [britely=URL width=SIZE]
1. Paste it into your post, and you are done!
